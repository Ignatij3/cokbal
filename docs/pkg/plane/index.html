type Plane ¶

Plane represents a set of all fractions which would be processed. It is defined as first quarter of circle with radius Radius.

func (Plane) GetResults ¶ 

func (p Plane) GetResults() []uint64

GetResults returns slice with total occurrences of elements of continued fractions in set plane. If calculations are running, the function returns nil.

func (Plane) NotifyOnFinish ¶ 

func (p Plane) NotifyOnFinish() <-chan struct{}

NotifyOnFinish returns channel which would receive signal when all calculations have finished. After receiving signal, it is safe to read obtained data.

func (*Plane) Start ¶ 

func (p *Plane) Start()

Start initializes calculations with given concurrent worker amount. Start does not block. To know, when have calculations finished, use NotifyOnFinish. If calculations are already running, Start does nothing. The function also calls interrupt listener, which will intercept one keyboard interrupt to correctly stop calculations.

func (*Plane) Stop ¶ 

func (p *Plane) Stop()

Stop prematurely stops calculations. When Stop exits, it is safe to read obtained data.

func (*Plane) TryToRestoreState ¶ 

func (p *Plane) TryToRestoreState() error

TryToRestoreState tries to load configuration for plane of specific radius. If error is returned, it might be *PathError or custom error with no type.
